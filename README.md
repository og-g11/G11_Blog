cookie-ს სახით. დამახსოვრებული ინფორმაციით შესული მომხმარებელი login გვერდზე შესვლის დროს უნდა
გადამისამართდეს მთავარ გვერდზე, როგორც მომხმარებელი. logout-ის შემთხვევაში ეს არ უნდა მოხდეს.
